## Usage

Use the version hosted [here](https://ivanthetricourne.github.io/project-arwing/index.html)--more than likely to be out of date but still works.

Alternatively, with `npm`:

```
npm install
npm start
```

## Development

```
npm install
npm run elm:live
```
